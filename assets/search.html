Implementation Details

Your existing jQuery code has been integrated with the following functionality:

  • Clicking the search icon toggles the search popup
  • When the search popup is active, the body is locked from scrolling
  • The mobile navigation wrapper is hidden when search is active
  • All animations and transitions are smooth and responsive
// Your existing jQuery code
if ($(".search-toggler").length) {
  $(".search-toggler").on("click", function (e) {
    e.preventDefault();
    $(".search-popup").toggleClass("active");
    $(".mobile-nav__wrapper").removeClass("expanded");
    $("body").toggleClass("locked");
  });
}
